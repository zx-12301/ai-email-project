## 📋 优化内容

### 1️⃣ 工具栏样式同步收件箱

**优化前**:
- ❌ 按钮太宽（`px-3 py-1.5 text-sm`）
- ❌ 图标太大（`w-4 h-4`）
- ❌ 间距太大（`gap-1.5`）
- ❌ 内边距太大（`p-4`）

**优化后**:
- ✅ 按钮紧凑（`px-2.5 py-1.5 text-xs`）
- ✅ 图标适中（`w-3.5 h-3.5`）
- ✅ 间距合理（`gap-1`）
- ✅ 内边距适中（`px-4 py-2`）

### 2️⃣ 样式对比

**收件箱样式** (标准):
```tsx
<button className="flex items-center gap-1 px-2.5 py-1.5 text-xs text-slate-700 hover:bg-slate-100 rounded">
  <Icon className="w-3.5 h-3.5" />
  按钮文字
</button>
```

**星标页优化后** (同步收件箱):
```tsx
<button className="flex items-center gap-1 px-2.5 py-1.5 text-xs text-slate-700 hover:bg-slate-100 rounded">
  <Icon className="w-3.5 h-3.5" />
  按钮文字
</button>
```

### 3️⃣ 测试数据标记

**新增功能**:
- ✅ 测试数据邮件带蓝色"测试"标签
- ✅ 标签样式：`bg-blue-100 text-blue-600`
- ✅ 字体大小：`text-[9px]`
- ✅ 位置：发件人名称右侧

## 🔧 技术实现

### 样式同步

**按钮样式**:
```tsx
// 统一使用收件箱样式
className="flex items-center gap-1 px-2.5 py-1.5 text-xs text-slate-700 hover:bg-slate-100 rounded transition-colors"
```

**图标尺寸**:
```tsx
// 统一使用 3.5x3.5
className="w-3.5 h-3.5"
```

**容器内边距**:
```tsx
// 统一使用 px-4 py-2
className="border-b border-slate-200 px-4 py-2 bg-white"
```

### 测试数据标记

**条件渲染**:
```tsx
{email.isTest && (
  <span className="px-1 py-0.5 text-[9px] bg-blue-100 text-blue-600 rounded">
    测试
  </span>
)}
```

**布局**:
```tsx
<div className="flex items-center gap-1">
  <span>{email.from}</span>
  {email.isTest && <span>测试</span>}
</div>
```
