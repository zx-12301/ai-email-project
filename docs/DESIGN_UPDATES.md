| 用途 | 颜色值 | Tailwind 类 |
|------|--------|-------------|
| 主背景 | #F8FAFC | `bg-slate-50` |
| 侧边栏 | #F1F5F9 | `bg-slate-100` |
| 内容区 | #FFFFFF | `bg-white` |
| 主色 | #2563EB | `text-blue-600` |
| 边框 | #E2E8F0 | `border-slate-200` |
| 文字主色 | #1E293B | `text-slate-900` |
| 文字次要 | #64748B | `text-slate-500` |
| 未读背景 | #EFF6FF | `bg-blue-50/50` |

### 圆角规范
- 按钮：`rounded` (4px)
- 卡片：`rounded-lg` (8px)
- 下拉菜单：`rounded-lg` (8px)

### 间距规范
- 小组件：`p-2` (8px)
- 中等组件：`p-3` (12px) / `p-4` (16px)
- 大组件：`p-6` (24px)

### 字体大小
- 超小：`text-xs` (12px)
- 小：`text-sm` (14px)
- 中：`text-base` (16px)
- 大：`text-lg` (18px)

## 💡 设计原则

1. **一致性** - 所有页面使用相同的设计语言和组件
2. **清晰度** - 信息层次分明，重要内容突出
3. **易用性** - 操作直观，反馈及时
4. **美观性** - 简洁现代，符合企业邮箱定位
5. **响应式** - 适配不同屏幕尺寸
