## 🔧 技术栈

### 前端
- **框架**: React 18 + TypeScript
- **构建**: Vite 5
- **路由**: React Router v6
- **样式**: Tailwind CSS 3
- **图标**: Lucide React

### 后端
- **框架**: NestJS
- **语言**: TypeScript
- **数据库**: PostgreSQL (待连接)

### AI 服务
- **接口**: 已预留 `/api/ai/*` 端点
- **功能**: 智能写信/回复/分类/检测
